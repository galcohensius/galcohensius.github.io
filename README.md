# Gal Cohensius - Personal Website

This website, developed as a response to the first daily challenge by [התלתל](https://x.com/taltimes2/status/1875879963393413514).

The challenge was to:
1. Host a personal website.
2. Include a domain or use GitHub Pages.
3. Deploy automatically after each commit.
4. Follow accessibility and mobile-friendly design principles.

## Features
- About section detailing my expertise in game algorithms.
- Contact section with a working form powered by Formspree.
- Responsive design for mobile, tablet, and desktop.
- Integrated links to GitHub, LinkedIn, StackExchange, and Google Scholar.

## View the Site
This website is hosted on [GitHub Pages](https://galcohensius.github.io/).

## Technical Details
- **HTML5**: For creating the structure of the website.
- **CSS3**: For styling and ensuring a responsive design.
- **Formspree**: For handling the contact form submissions.
- **GitHub Pages**: For hosting the site with free deployment.
- **Font Awesome**: For icons in the "About" section.
